## usage
```bash
git clone https://github.com/zxj5470/personal-bashrc.git
# git clone git@github.com:zxj5470/personal-bashrc.git
cd personal-bashrc
cp .bashrc ~
cp .inputrc ~
source ~/.bashrc
```



## add .inputrc to ignore UpperCase or lowerCase in command line
```bash
git add .inputrc -f
```
